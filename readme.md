## 💻 Instalação
1. ```git clone``` no projeto.
2. Execute o comando ```yarn``` ou ```npm install``` no diretório raiz do projeto.
3. Execute o comando ```yarn dev``` ou ```npm run dev```

## 👩‍💻 Tecnologias

 - Socket.io
 - Express
 - EJS
